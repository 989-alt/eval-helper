# 평어 도우미 (eval-helper)

교사용 평어 생성 도구. **누구나** — 어떤 AI를 쓰든, AI를 아예 쓰지 않든 — 교과 평어·창의적 체험활동·행동발달 문장을 쉽게 만들 수 있습니다.

🔗 https://989-alt.github.io/eval-helper/

## 특징
- **내장 무료 엔진**: API 키 없이 즉시 교과·창체 평어 생성(표현 풀+규칙 기반).
- **내 AI(선택)**: Google / OpenAI / Anthropic 키를 넣으면 더 자연스러운 AI 생성 + 행동발달.
- **모델 실시간 조회**: 키를 넣으면 그 시점에 실제 제공되는 모델만 표시하고, 모델명 옆에 비용 티어(무료 추천 / $20 라이트 / $100 헤비)를 자동으로 표시합니다. 목록이 낡지 않습니다.
- **교과 모드 B**: 학생별 점수(성취수준)를 표나 엑셀로 받아 **전 과목·평가요소 평어를 한 번에** 생성.
- **앱 생성 엑셀 양식**: 정의한 과목·평가요소·학생에 맞춰 엑셀 양식을 만들어 주고, 채워서 업로드하면 그대로 읽습니다.
- **클릭 복사**, 한글(.doc) / 텍스트 내보내기.

## 평어 규칙(교과·창체)
명사형 종결(~함/~임), 특수문자·영어 금지(마침표·쉼표만, 단 4·19 등 핵심 개념의 숫자·가운뎃점 허용), 기업·제품명 금지, 모든 평어 서로 다르게, 성취수준별 어조 차등.

## 구조
빌드리스 정적 앱(ES 모듈 바닐라 JS + Tailwind/SheetJS CDN). 모든 처리는 브라우저에서만 이루어지며 입력과 API 키는 서버로 전송되지 않습니다. GitHub Pages(main 루트)로 배포.

자세한 설계는 [docs/SPEC.md](docs/SPEC.md) 참고.
